# Development CLI



## Usage
* When closing shell with the exit linux command

## Development
* bin/build.sh - builds the docker image
* bin/run.sh - runs the docker image as a container

### Prerequisite
Install [Powerline](https://github.com/powerline/fonts) fonts

### MacOS
```
brew install neovim
```
